# ecommerce_app

How to run the app
```

flutter pub get
flutter run

```

## Features

  * [x] Sign in with email & password
  * [x] Anonymous Sign in
  * [x] Firebase Storage
  * [x] Add items to cart
  * [x] Add items to liked page
  * [x] Logout user
